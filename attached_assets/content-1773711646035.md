# What is Koyeb?

Koyeb is a **developer-friendly serverless platform** designed to let businesses and developers easily deploy reliable and scalable applications globally. The platform was created by [Cloud Computing Veterans](https://www.koyeb.com/about) and is financially backed by industry leaders. _[Learn more about our team and investors](https://www.koyeb.com/about)._

Koyeb provides a unified way to run inference, execute code in isolated sandboxes, and **deploy a variety of [services](https://www.koyeb.com/docs/reference/services)** including full web applications, [APIs](https://www.koyeb.com/tutorials/tag/api), [MCP servers](https://www.koyeb.com/tutorials/tag/mcp), [AI Agents](https://www.koyeb.com/tutorials/tag/agents), and background workers.

### Developer-friendly and cloud-agnostic [Permalink for this section](https://www.koyeb.com/docs\#developer-friendly-and-cloud-agnostic)

Deploy on CPUs, GPUs, or accelerators without changing your code. Koyeb abstracts compute so workloads run on any hardware, in any region, across clouds. Use [Docker containers](https://www.koyeb.com/docs/build-and-deploy/prebuilt-docker-images) or deploy from [Git repositories](https://www.koyeb.com/docs/build-and-deploy/build-from-git).

### Built-in scaling, reliability, and automation [Permalink for this section](https://www.koyeb.com/docs\#built-in-scaling-reliability-and-automation)

Koyeb provides an efficient way to run apps without the heavy-duty tasks of building **resilient and scalable infrastructure**. All of the Services you deploy on Koyeb come with native scaling and autohealing features to completely abstract the server layer. With Koyeb’s Scale-to-Zero feature, your **[Instances](https://www.koyeb.com/docs/reference/instances) can automatically scale down to zero** when there is no incoming traffic, allowing you to optimize costs. New Service deployments are dramatically simplified thanks to a [full-service mesh](https://www.koyeb.com/blog/what-is-a-service-mesh) providing secure inter-service connections and automatic Service discovery. _Learn more about Koyeb’s [autoscaling](https://www.koyeb.com/docs/run-and-scale/autoscaling) and [Scale-to-Zero](https://www.koyeb.com/docs/run-and-scale/scale-to-zero) features._

### High performance and security [Permalink for this section](https://www.koyeb.com/docs\#high-performance-and-security)

The core of the platform has been built with security and performance in mind. Applications transparently run in fast and secure [Cloud Hypervisor](https://www.cloudhypervisor.org/) microVMs on top of bare metal servers with high-end CPUs or GPUs. _[Learn more about our GPUs and benchmarks](https://www.koyeb.com/docs/hardware/gpu-benchmarks)_

### Global deployment options [Permalink for this section](https://www.koyeb.com/docs\#global-deployment-options)

Deploy globally. The Koyeb Edge Network provides global CDN and load-balancing. Choose instance configurations based on CPU, GPU, and RAM requirements.

### Get started for free [Permalink for this section](https://www.koyeb.com/docs\#get-started-for-free)

Start with a **`free` Instance**: 512MB of RAM, 0.1 vCPU, and 2GB of SSD. [Sign up now](https://app.koyeb.com/auth/signup) and **deploy one of our [Quickstart Guide Applications](https://www.koyeb.com/docs/deploy) in seconds** to see how Koyeb works with your favorite technologies.

## The Koyeb experience [Permalink for this section](https://www.koyeb.com/docs\#the-koyeb-experience)

Koyeb provides a unified experience to run inference, sandboxes, and production workloads on any hardware, anywhere.

### Core features at a glance [Permalink for this section](https://www.koyeb.com/docs\#core-features-at-a-glance)

The Koyeb platform provides:

- An **easy-to-use web interface** to manage all your app deployments: [create your account for free](https://app.koyeb.com/auth/signup).
- Support for **multiple types of Services**, including full web applications, AI agent sandboxing, model training and fine-tuning, model serving, static and dynamic sites, APIs, and background workers.
- Full support for Docker containers.
- Git-driven deployment to **build and deploy native code** in Ruby, Node.js, Java, Python, Clojure, Scala, Go, Rust, PHP, or with a Dockerfile present in the repository.
- A high-performance **Edge Network with a global CDN** and powerful **load-balancing across zones**.
- Full service mesh and discovery to **deploy secure microservices in seconds**.
- Transparent deployment in fast, secure MicroVMs.
- The [Koyeb CLI](https://www.koyeb.com/docs/build-and-deploy/cli/installation) (Command Line Interface) to manage resources and automate directly from your terminal.
- An easy-to-use REST API for interacting with Koyeb programmatically. _Check out the [API documentation](https://www.koyeb.com/docs/reference/api)._
